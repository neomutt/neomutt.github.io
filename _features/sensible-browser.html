 Sensible-Browser Patch

Make the file browser behave

1. Patch

To check if Mutt supports “sensible-browser”, look for “patch-sensible-browser” in the mutt version. See: Section 1.2, “Mutt Patches”.

Dependencies:

  • mutt-1.5.24

This patch is part of the NeoMutt Project.

2. Introduction

When using the file/folder browser, the selection isn't always where you would expect. When you navigate to '..' (parent folder), the selection should show the folder you used to be in.

3. Muttrc

None

  • Should only select folder when folder is sorted alphabetically
