 Attached Patch

Prevent missing attachments

1. Patch

To check if Mutt supports “ attached”, look for “ patch-attached”in the mutt version. See: Section 1.2, “Mutt Patches”.

Dependencies:

  • mutt-1.6.2

This patch is part of the NeoMutt Project.

2. Introduction

If you use the word attachment in your mail, but don't actually attach anything, this patch will ask you if you're sure you want to send the mail.
